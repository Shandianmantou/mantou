# 🚀 5分钟免费上线指南

## 最简单的方法：Vercel（完全免费）

### ✅ 已为你准备好的文件
- ✅ `api/` 文件夹 - 所有API接口（Vercel自动识别）
- ✅ `vercel.json` - Vercel配置文件
- ✅ `admin.html` - 已更新，自动适配本地/线上

## 📝 详细步骤

### 第1步：注册 GitHub 账号（2分钟）
1. 访问 https://github.com
2. 点击右上角 "Sign up"
3. 填写用户名、邮箱、密码
4. 完成邮箱验证

### 第2步：上传代码到 GitHub（3分钟）

**方法A：网页上传（最简单）**

1. 登录 GitHub，点击右上角 "+" → "New repository"
2. 仓库名：`mantou-music`（随便起名）
3. 选择 **Public**（公开）
4. **不要**勾选任何选项（README、.gitignore等）
5. 点击 "Create repository"
6. 在新页面点击 "uploading an existing file"
7. **拖拽项目文件夹里的所有文件**到页面
8. 点击绿色按钮 "Commit changes"

完成！代码已上传。

### 第3步：部署到 Vercel（1分钟）

1. 访问 https://vercel.com
2. 点击 "Sign Up" → 选择 "Continue with GitHub"
3. 授权 GitHub 账号给 Vercel
4. 登录后，点击 "Add New Project"
5. 在列表中找到你的 `mantou-music` 仓库
6. 点击 "Import"
7. 直接点击 **"Deploy"**（所有设置都是自动的）
8. 等待 1-2 分钟...

### 第4步：获得网址 ✨

看到 ✅ "Congratulations!" 后，你会得到一个网址：
```
https://mantou-music.vercel.app
```

**这就是你的网站！** 分享给朋友就可以访问了！

## 🎯 之后的使用

### 更新网站内容
1. 在 `admin.html` 添加/编辑内容
2. 或者在 GitHub 网页上直接编辑文件
3. Vercel 会自动重新部署（几分钟内更新）

### 访问地址
- **主页**：`https://你的项目名.vercel.app`
- **管理后台**：`https://你的项目名.vercel.app/admin.html`

## ⚠️ 重要提示

### 数据存储
- Vercel 的临时存储可能会重置
- 如果需要永久保存，建议：
  1. 定期手动备份（在管理界面导出数据）
  2. 或使用数据库（MongoDB Atlas 免费版）

部署成功后，可以在 Vercel 设置中添加你自己的域名！
