# 🚀 部署指南 - 免费上线方案

## 方案一：Vercel（推荐，完全免费）⭐

### 为什么选择 Vercel？
- ✅ **完全免费**（个人项目够用）
- ✅ **自动部署**（代码推送到GitHub就自动更新）
- ✅ **全球CDN加速**（朋友访问速度快）
- ✅ **自带域名**（也可以绑定自己的域名）
- ✅ **零配置**（不需要懂服务器）

### 快速部署（5分钟搞定）

#### 1. 注册 GitHub 账号
- 访问 https://github.com
- 注册账号（免费的）

#### 2. 上传代码到 GitHub
1. 在 GitHub 上新建一个仓库（Repository）
2. 把你项目的所有文件上传到仓库
3. 或者用 Git 命令（如果会用的话）

#### 3. 连接 Vercel
1. 访问 https://vercel.com
2. 用 GitHub 账号登录
3. 点击 "Add New Project"
4. 选择你的仓库
5. 点击 "Deploy"（部署）

**完成！** 会给你一个网址，比如：`your-project.vercel.app`

### 优点

5. 自动检测 Node.js，一键部署

### Vercel部署需要调整
因为 Vercel 使用 Serverless Functions，需要把后端代码改成 API Routes 格式。

我已经为你准备好了：

### 数据持久化
免费方案数据可能不会永久保存，如果需要：
1. 使用数据库（MongoDB Atlas 免费版）
2. 或者定期备份 `data/` 文件夹
